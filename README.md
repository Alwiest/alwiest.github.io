# alwiest.github.io
# Сайт-портфолио

Статический сайт-портфолио для учебных лабораторных работ, созданный на базе MkDocs и развернутый через GitHub Pages.

 **Сайт доступен по адресу:** [https://alwiest.github.io/](https://alwiest.github.io/)

---

## Оформление и тема

### Выбранная тема: **Bootswatch darkly**

**Обоснование выбора:**
1.  **Тёмная схема по умолчанию** — снижает нагрузку на глаза при длительной работе с кодом и текстом.
2.  **Минимальная настройка** — не требует сложной конфигурации палитры.
3.  **Читаемость** — контрастные цвета обеспечивают удобное восприятие текста.
4.  **Стиль** — темная тема воспринимается современно и лаконично.

---

## Локальный запуск

1.  **Клонирование репозитория:**
    ```bash
    git clone https://github.com/Alwiest/alwiest.github.io.git
    cd alwiest.github.io
    ```

2.  **Создание и активирование венв:**
    ```
    python -m venv venv
    .\venv\Scripts\Activate.ps1
    ```

3.  **Установка зависимостей:**
    ```bash
    pip install mkdocs mkdocs-bootswatch
    ```

4.  **Запуск локального сервера:**
    ```bash
    cd src
    mkdocs serve
    ```
    Сайт будет доступен по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
