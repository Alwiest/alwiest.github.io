# Лабораторная работа №5
## Тема: Предсказание цен на недвижимость (Регрессия)

### 1. Цель работы
Построить регрессионные модели для предсказания стоимости домов в округе Кинг (Сиэтл). Сравнить эффективность линейной регрессии, случайного леса и градиентного бустинга. Оптимизировать гиперпараметры моделей для минимизации ошибок MAE, RMSE

### 2. Ход работы
1.  Загружены данные из Excel (`predict_house_price_training_data.xlsx`).
2.  Проведен анализ данных: выявлено 15 129 объектов обучения и 6 484 теста. Пропусков нет.
3.  Целевая переменная: `Целевая.Цена`.
4.  Обучены базовые модели: Linear Regression и RandomForestRegressor.
5.  **Самостоятельная работа:**
    *   Проанализирована важность признаков (`feature_importances_`). Удалены наименее значимые признаки (важность < 0.01).
    *   Выполнен подбор гиперпараметров для Random Forest через `GridSearchCV`.
    *   Реализована модель CatBoostRegressor для повышения точности.

### 3. Результаты
Сравнение моделей по метрике RMSE (Root Mean Squared Error, чем меньше, тем лучше):

| Модель | RMSE ($) | MAE ($) |
| :--- | :--- | :--- |
| Linear Regression | ~201 883 | ~126 852 |
| Random Forest (Base) | ~137 104 | ~70 712 |
| Random Forest (Tuned) | ~135 000 | ~69 000 |
| **CatBoost** | **~125 000** | **~65 000** |


Модель CatBoost показала наилучший результат, снизив ошибку предсказания по сравнению с линейной регрессией более чем на 35%.
