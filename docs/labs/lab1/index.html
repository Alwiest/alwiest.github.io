Лабораторная работа 1

Тема: Создание и развертывание статического сайта на базе MkDocs

Цель работы

  • Освоить процесс создания статического сайта с использованием генератора документации MkDocs.
  • Научиться организовывать структуру документации проекта.
  • Развернуть сайт с использованием GitHub Pages.

Задание

  1. Создать публичный репозиторий на GitHub.
  2. Настроить GitHub Pages для публикации из каталога /docs.
  3. Создать и настроить виртуальное окружение Python.
  4. Установить и настроить MkDocs.
  5. Реализовать структуру сайта. В моем случае: Главная, «Об авторе», раздел «Лабораторные работы»
  6. Выбрать и подключить тему оформления.
  7. Выполнить сборку и публикацию сайта.

Ход работы

1. Настройка окружения

python -m venv venv
.\venv\Scripts\Activate.ps1
pip install mkdocs mkdocs-bootswatch

2. Инициализация проекта

mkdocs new src
cd src

3. Конфигурация mkdocs.yml

site_name: Сайт-портфолио Alwiest

theme:
  name: darkly

nav:
  - Главная: index.md
  - Об авторе: about.md
  - Лабораторные работы:
      - Лабораторная 1: labs/lab1.md
      - Лабораторная 2: labs/lab2.md
      - Лабораторная 3: labs/lab3.md

4. Сборка и публикация

mkdocs build -d ../docs git add . git commit -m "final fixes" git push
